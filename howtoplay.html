How to Play Project Nova

Joining is simple! Follow these steps to get started:

  1. Download and install RadminVPN.
  2. Join our RadminVPN network: [Your Network Name Here] (Password: [Your Network Password Here]).
  3. Download our custom client here. (You'll need to upload your client somewhere and replace this '#' with the actual download link, or provide more detailed instructions.)
  4. Launch the client and connect to: [Your VM's RadminVPN IP]:[Your Server Port] (e.g., 26.x.x.y:2050).
  5. If you encounter any issues during the setup process, please join our Discord server for direct assistance from staff and community members!

Important Notes:

  • Ensure RadminVPN is running before launching the client.
  • Your server IP will be the RadminVPN IP of the machine running the Project Nova server.
  • The server port (e.g., 2050) must match what your server is configured to use.
